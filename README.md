## **🚀 Quick Start**  

1️⃣ Clone the repo:  
```sh
git clone https://github.com/Tusharxhub/Ai-Agent.git
cd Ai-Agent
```
  
2️⃣ Install dependencies:  
```sh
npm install
```

3️⃣ Run the app:  
```sh
npm start
```

🚀 *Your AI Agent is now up and running!* 🎉
